# mdn-js-book-demo-server


## Requirement

- node > v0.6.11
- redis

## Setup

    $ git clone git@github.com:hagino3000/mdn-js-book-demo-server.git
    $ cd mdn-js-book-demo-server
    $ npm install

## Start Server

    $ make run

## Show Index Page

After start server, open http://localhost:3002/index.html in your browser.
